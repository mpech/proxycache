# Proxycache

Caches requests and serve the further requests from cached file


## Usage

```
pnpm install
npm run cli -- -m '3004->http://127.0.0.1:3005' -m '3006->http://127.0.0.1:3007' -v
# or if you dev
npm start -- -m '3004->http://127.0.0.1:3005' -m '3006->http://127.0.0.1:3007' -v
```
